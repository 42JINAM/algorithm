# Non-primitive Data structure

## Linear

A structure that connects data elements sequentially.

### Array
### Linked list
### Stack
### Queue
### Heap

## Non-Linear

A structure with hierarchical or many to many connections between data elements.

### Tree
### Graph

## Abstract data types (ADT)

ADT is a blueprint that defines what operations a data structure must support, without specifying how they are implemented.


# Why do we have to learn about data structure? 

Program performance varies depending on the data structure used. By understanding how data structures work internally, we can choose the right one for our program design.
